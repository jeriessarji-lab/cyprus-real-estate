# Cyprus Luxury Properties — Deployment Guide
## How to go live on Vercel in 15 minutes (FREE)

---

## STEP 1 — Create your project files

Create a folder on your computer called: `cyprus-real-estate`

Inside it, create these 3 files exactly as named:
- `index.html`
- `package.json`
- `vite.config.js`

(File contents are in the separate code artifacts below)

---

## STEP 2 — Install Node.js (one time only)

1. Go to https://nodejs.org
2. Download the **LTS** version (green button)
3. Install it (just click Next → Next → Finish)

---

## STEP 3 — Build the project

Open a terminal / command prompt in your project folder and run:

```
npm install
npm run build
```

This creates a `dist` folder — that's your finished website.

---

## STEP 4 — Deploy to Vercel (FREE)

1. Go to https://vercel.com and sign up for a FREE account
2. Click **"Add New Project"**
3. Click **"Upload"** (no GitHub needed)
4. Drag and drop your `dist` folder
5. Click **Deploy**

✅ Your site will be LIVE in 60 seconds at a free Vercel URL like:
`https://cyprus-real-estate.vercel.app`

---

## STEP 5 — Connect your own domain (optional, ~€12/year)

1. Buy a domain at https://namecheap.com
   Suggested names:
   - cyprusluxuryproperties.com
   - sarji-cyprus-realty.com
   - cyprusnewproperties.com

2. In Vercel → your project → Settings → Domains
3. Type your domain name and click Add
4. Vercel will show you DNS settings to copy into Namecheap
5. Done — live on your domain within 1 hour

---

## STEP 6 — Connect Google Analytics (FREE traffic tracking)

1. Go to https://analytics.google.com
2. Create a free account → New Property
3. Copy your **Measurement ID** (looks like: G-XXXXXXXXXX)
4. Add this line inside the `<head>` tag in your `index.html`:

```html
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
</script>
```

Replace G-XXXXXXXXXX with your actual ID.

Now you can see:
- How many visitors per day
- Which countries they come from (Middle East, Europe, etc.)
- Which pages/properties they visit most
- How long they stay

## UPDATING YOUR LISTINGS

To add/edit properties after the site is live:
1. Use the Admin panel inside the app (+ Add Property button)
2. Re-run `npm run build`
3. Go to Vercel → your project → Deployments → Upload new `dist` folder
